# 👁️ Gabriel's Cyberpunk Red Rice

Um setup minimalista, rápido e agressivo para Arch Linux (Hyprland), focado em performance para jogos e estética dark/neon.

![Wallpaper Preview](https://i.imgur.com/example.png) *(Coloque um print do seu desktop aqui!)*

## 🚀 Destaques
- **Window Manager:** [Hyprland](https://hyprland.org/) (Bordas Neon Animadas)
- **Barra:** [Waybar](https://github.com/Alexays/Waybar) (Design minimalista flutuante)
- **Terminal:** [Kitty](https://sw.kovidgoyal.net/kitty/) (Otimizado para Gaming, low latency)
- **Cores:** [Pywal](https://github.com/dylanaraps/pywal) (Sincronia total com o Wallpaper)
- **Apps Estéticos:** `tty-clock` e `lavat` sincronizados com o tema.

## 🛠️ Comandos Customizados
Adicionados ao meu `.bashrc`:
- `clock`: Relógio grande com as cores do tema.
- `lava`: Luminária de lava com cores dinâmicas.

## 🎨 Cores (Pywal)
O sistema gera automaticamente a paleta de cores baseada no wallpaper atual, aplicando bordas neon nas janelas e gradientes no visualizador de áudio (Cava).

## 📦 Instalação
1. Clone este repositório:
   ```bash
   git clone https://github.com/seu-usuario/dotfiles.git
   ```
2. Copie os arquivos para a sua `.config`:
   ```bash
   cp -r .config/* ~/.config/
   ```
